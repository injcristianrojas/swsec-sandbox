Getting started

Bricks is built using PHP and MySQL. Bricks are classified into three different sections: login pages, file upload pages and content pages. Each section has a number of pages which posses some security issues.

What all are needed
  • Bricks:
    Latest version of Bricks can be downloaded for free from the download page.

  • PHP and MySQL:
    uWAMP portable version is a good option for Windows users. (Used in this guide)
    XAMPP is also good and are avaible for Linux, Windows, Macintosh and Solaris.

  • Browser
    OWASP Mantra is the best option. (Used in this guide)

  • Other software:
    7-Zip, EXIFeditor, SQLMap
